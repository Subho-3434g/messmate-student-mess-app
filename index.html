Firebase setup needed One time

To make login and cloud sync live for everyone, add your Firebase web config in firebase-config.js, enable Email/Password Auth, and enable Firestore.

1. Create a Firebase project. 2. Enable Authentication: Email/Password. 3. Enable Firestore Database. 4. Paste your web app config into firebase-config.js. 5. Add firestore.rules, then push/deploy again.
